# NexusEcom ⚡ Roadmap de Implementação

## Fase 1: Fundação & Identidade (100% Concluída)
- [x] **Arquitetura Base:** Laravel 11 + Livewire 3 + Tailwind (TALL Stack).
- [x] **Banco de Dados:** PostgreSQL `nexusecom` com Multi-tenancy isolado.
- [x] **Autenticação:** Sistema de login premium com Dark Mode unificado.
- [x] **Identidade Visual:** Logo oficial SVG (Raio/Escudo) e layout High-End.

## Fase 2: Gestão Corporativa & Fiscal (Em Andamento)
- [x] **Hub de Empresas:** Unificação de dados cadastrais e integrações.
- [x] **Campos Fiscais:** Implementação de Razão Social, Nome Fantasia, Apelido, CNPJ e NCM.
- [x] **Certificado Digital:** Estrutura para upload e armazenamento seguro de Certificado A1 e Senha.
- [x] **Branding Individual:** Upload de Logos por empresa com processamento de imagem.
- [ ] **WMS Multiestoque:** Lógica de armazéns isolados ou compartilhados.
- [ ] **Exportação Contábil:** Geração de ZIP mensal de XMLs para contabilidade.

## Fase 3: Conectores Marketplace (Próxima)
- [ ] **OAuth Mercado Livre:** Conexão oficial via API para gestão de contas.
- [ ] **Sincronização de Anúncios:** Importação da base do Meli para o NexusEcom.
- [ ] **Monitor de Vendas:** Painel em tempo real de pedidos e status logístico.
